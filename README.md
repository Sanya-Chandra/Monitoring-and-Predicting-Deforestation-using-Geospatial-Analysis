# Monitoring and Predicting Deforestation Using Geospatial Analysis

This project aims to monitor, analyze, and predict deforestation patterns in India using Geospatial Analysis, Remote Sensing, and Machine Learning techniques. The system utilizes environmental and geographical factors such as NDVI (Normalized Difference Vegetation Index), rainfall, temperature, land location, and vegetation data to identify regions affected by deforestation.

By processing satellite imagery and spatial datasets, the project helps in detecting forest loss, analyzing environmental changes, and forecasting areas that are at high risk of future deforestation. The results are visually represented on interactive maps for easier understanding and decision-making.

The system is designed to support environmental monitoring, forest conservation planning, climate studies, and sustainable resource management through data-driven insights and predictive analytics.
